# ignite-challenge-02

## Getting Started

```bash
# Cloning the repository
git clone https://github.com/trickvg/ignite-challenge-02

cd ignite-challenge-02
```

### Install dependencies and running application

##### Yarn

```bash
#Installing dependencies
yarn install

# Running app
yarn dev

# Running server
yarn server
```
##### NPM
```bash
#Installing dependencies
npm install

# Running app
npm run dev

# Running server
npm run server
```
